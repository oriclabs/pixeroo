Features

Everything images, in one place

Image Inspector

Dimensions, file size, MIME type, EXIF/GPS data, ICC profiles, alt text, DPI — all from a right-click.

Format Converter

Real pixel-level conversion between PNG, JPEG, WebP, AVIF, TIFF, BMP, QOI, ICO, SVG and more. Not just renaming.

QR Code Studio

Generate QR codes for URLs, WiFi, vCards. Read QR codes from any image on the page. Export as PNG or SVG.

Smart Downloader

Bulk download with filters, smart filenames, format conversion on-the-fly, ZIP export. Finds originals, not thumbnails.

Quick Editor

Resize, crop, rotate, flip, adjust brightness/contrast, apply filters — all without leaving the browser.

Color Extraction

Pick any pixel color, extract dominant palettes, copy as HEX/RGB/HSL. Perfect for designers.

SVG Tools

Render SVG to raster at any resolution, inspect structure, view source. No other extension does this.

Clipboard Magic

Copy any image as PNG to clipboard, copy as base64, paste images into the editor from clipboard.

Accessibility Audit

Find missing alt text, oversized images, broken images, and performance issues on any page.

How It Works

Runs entirely in your browser

G

Grab

Capture screenshots, grab page images, select regions, pick colors, drop files, paste from clipboard — all from your browser.

A

Alter

Edit, annotate, filter, compress, convert, draw, create mockups, memes, certificates, QR codes, collages — all locally.

Z

Zone

Focus on what matters — crop, safe zones, social platform frames, drag to position, side-by-side compare, pixel inspect.

O

Output

Export PNG, JPEG, WebP, SVG, GIF. Copy to clipboard, download as ZIP, save to library — images never leave your device.
