Changelog

v0.1.0 March 2026 Initial Release

Every pixel. One toolkit.

Side Panel -- Browse page images with 5 view modes (tiles, medium, large, details, names), sort, filter, type badges, overlay inspector with EXIF, bulk download

Colors Tab -- Eyedropper with magnifier (captureVisibleTab), favorites library (synced), session picks, export as CSS/JSON/text

Screenshot -- Capture visible tab, copy to clipboard, save as PNG

Toolkit (9 modes):

  • Edit -- resize, interactive crop (5 ratios), rotate, flip, brightness/contrast/saturation/hue, filters, annotate (rect/arrow/text/redact), watermark, vignette, denoise, rounded corners, border, tiling, color blindness simulation, CMYK preview, histogram, levels adjustment, pixel art, sprite slicer, steganography detector, channel separation, color replace, background remove, padding, split, social media presets, favicon preview, ASCII art, reverse image search, metadata strip, image to PDF
  • Convert -- batch format conversion (PNG/JPEG/WebP/AVIF/BMP/ICO/TIFF/QOI), quality controls, batch resize, compression optimizer, metadata strip
  • Store Assets -- icon/screenshot generation for Google Play, Apple App Store, Chrome Web Store, Edge Add-ons, Firefox Add-ons, Microsoft Store
  • Info -- file info, DPI/PPI, EXIF, JPEG structure analysis, image hash (SHA-256 + perceptual), base64 export
  • QR Code -- generate with custom colors/size/ECC, presets (URL/WiFi/email/phone/vCard), export PNG/SVG, QR reader (WASM)
  • Colors -- eyedropper on local images, dominant palette extraction (k-means), HEX/RGB/HSL
  • SVG -- inspect structure, render to raster at custom resolution, view/copy source
  • Compare -- side-by-side, pixel diff with percentage
  • OCR -- text extraction (WASM engine)

Popup -- Instant QR for current page, Page Images + Toolkit launchers

Generators -- Gradient, checkerboard pattern, placeholder images from scratch

Privacy -- 100% offline, zero CDN dependencies, zero network calls, 7 minimal permissions, open source

Cross-browser -- Chrome, Edge, Firefox, PWA
