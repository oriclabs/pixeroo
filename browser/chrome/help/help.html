Pixeroo Help

Every pixel. One toolkit. v0.1.0

Getting Started

Pixeroo has three surfaces:

  • Popup -- Click the Pixeroo icon. Instant QR for current page + launchers for Page Images and Toolkit.
  • Side Panel -- Browse, inspect, and download images from the current webpage. Pick colors with the eyedropper. Capture screenshots.
  • Toolkit -- Full-page tab with 9 tool modes for editing, converting, and creating images from local files.

Everything runs 100% offline. Your images never leave your device.

Side Panel: Images Tab

Open the side panel from the popup ("Page Images" button) or press Alt+P.

Gallery Views

5 view modes in the toolbar: Tiles, Medium, Large, Details, Names. Switch between them to find the view that works best.

Sort & Filter

Sort by page order, file size, dimensions, type, or name. Filter by format (JPEG, PNG, WebP, SVG, GIF, AVIF).

Image Overlay

Click any image to open the overlay inspector showing:

  • Preview, filename, type, dimensions, displayed size, file size, alt text, URL
  • EXIF data (camera, GPS, date) -- loaded asynchronously
  • Actions: Save As (6 formats), Copy as PNG, Download, Extract Colors, Read QR Code

Use arrow keys to navigate between images. Press Escape to close.

Downloading

Select images with checkboxes, then click "Selected" or "All" in the bottom bar. Use the select toggle button to select/deselect all.

Favicons

Pixeroo detects favicon links (<link rel="icon">) and includes them in the gallery alongside regular images.

Side Panel: Colors Tab

Eyedropper

Click "Pick Color" to activate the eyedropper. A crosshair appears on the webpage with a magnifier showing the zoomed pixel. Click any pixel to pick its color. Press Escape to cancel.

Picked Colors

Every picked color appears in the list with its HEX and RGB values. Click the hex to copy. Click the heart to add to favorites.

Favorites

Favorite colors are stored permanently and synced across your devices. Right-click a favorite to remove it.

Export

Export all colors as CSS Variables, JSON, or plain text via the Export dropdown.

Side Panel: Screenshot

Click the camera icon to capture the visible tab. The screenshot appears in the panel where you can copy it to clipboard or save as PNG.

Toolkit: Edit Mode

Open from the popup ("Toolkit" button) or the home screen's "Edit" card. Drop, browse, or paste (Ctrl+V) a local image file.

Resize

Set width/height, lock aspect ratio, click Apply.

Crop

Choose a ratio (Free, 1:1, 4:3, 16:9, 3:2). Drag the crop area on the canvas. Drag corners to resize, drag inside to move. Click Apply Crop or Cancel.

Transform

Rotate left/right (90 degrees), flip horizontal/vertical.

Adjustments

Sliders for brightness, contrast, saturation, hue rotation.

Levels

Black point, white point, and gamma sliders for fine-grained tonal control.

Annotate

Draw rectangles, arrows, text, or redact (pixelate) regions. Choose color and line width.

Watermark

Enter text, set opacity, click Apply. Creates a diagonal tiled watermark across the entire image.

Export

Save in PNG, JPEG (85%), WebP, AVIF, BMP, or ICO format.

Toolkit: Convert Mode

Drop one or multiple files. Choose output format (8 options), set quality for lossy formats, optionally batch resize, and convert. Compression preview shows file sizes at different quality levels.

Toolkit: Store Assets Mode

Upload a source icon (ideally 1024x1024). Select target stores from the left nav. Click Generate to create all required icon and promo sizes. Click any asset card to download individually.

Supported: Google Play, Apple App Store, Chrome Web Store, Edge Add-ons, Firefox Add-ons, Microsoft Store.

Toolkit: Info Mode

Drop an image to inspect: file info, DPI/PPI, EXIF metadata, JPEG structure analysis, image hash (SHA-256 + perceptual), and copy as Base64 data URI.

Toolkit: QR Code Mode

Enter text or use a preset (URL, WiFi, email, phone, vCard). Customize error correction, pixel size, margin, and colors. Export as PNG, SVG, or copy to clipboard. Drop an image with a QR code to read it (requires WASM engine).

Toolkit: Colors Mode

Drop an image and click any pixel to pick a color. Extract dominant palette (2-12 colors) using k-means clustering. All values in HEX, RGB, HSL.

Toolkit: SVG Mode

Drop an SVG file to view it, inspect structure (width, height, viewBox, element count), render to raster (PNG/JPEG/WebP) at custom resolution, and copy SVG source.

Toolkit: Compare Mode

Drop Image A and Image B side by side. Click Diff to see pixel differences highlighted in red with a percentage. Click Swap to switch positions.

Toolkit: OCR Mode

Text extraction from images. Requires WASM engine to be built. Supports 11+ languages.

Keyboard Shortcuts

  • Alt+Q -- Generate QR for current page
  • Alt+P -- Open side panel
  • Ctrl+Z -- Undo (in editor)
  • Ctrl+Y -- Redo (in editor)
  • Ctrl+V -- Paste image into editor
  • Escape -- Close overlay / cancel eyedropper
  • Arrow Left/Right -- Navigate images in overlay

Customize shortcuts at chrome://extensions/shortcuts.

Settings

Click the gear icon (in popup, side panel, or toolkit). Available settings:

  • Theme (dark / light)
  • Default export format
  • Download subfolder name
  • QR default error correction
  • Side panel default view and sort
  • Reset disclaimer / reset all

Privacy

  • 100% offline processing via WebAssembly
  • Zero network calls, zero CDN dependencies
  • No data collection, analytics, or telemetry
  • No ads or affiliate link injection
  • 7 minimal permissions (activeTab, scripting, contextMenus, downloads, storage, sidePanel, clipboardWrite)
  • Open source at github.com/oriclabs/pixeroo
